v3-tokyo-editorial Jun 26, 2026 Latest

Editorial travel journal — Tokyo in Five

A complete redesign as a single-file editorial travel itinerary. Cormorant Garamond + DM Sans typography, warm paper tones with per-day accent colors, vertical timeline layout, sticky day navigation, and scroll-reveal animations. Fully self-contained — no frameworks, no build step.

Editorial Travel journal Cormorant Garamond Timeline layout Scroll animations Single file
Update log
  • Full 5-day Tokyo itinerary with 22 curated experiences across 4 districts
  • Vertical timeline layout with dot markers, hover states, and per-event color tags
  • Day-by-day sticky navigation with intersection observer active state
  • Scroll-reveal animation on summary section
  • Typography: Cormorant Garamond (display) + DM Sans (body) — editorial serif meets clean sans
  • Warm paper palette with per-day accent colors (rust, sage, lavender, clay)
  • Responsive design — works on mobile through desktop
  • Self-contained single HTML file, no external dependencies beyond Google Fonts
Design stance
  • Layout: Full-width hero → sticky day nav → timeline sections → summary stats
  • Typography: Cormorant Garamond for display (300/400/500) + DM Sans for body (300–700)
  • Color: Warm cream paper (#faf8f5), clay (#c4a882), rust (#b8705a), sage (#7a8f72), lavender (#8a7ab0)
  • Interaction: Scroll-triggered nav highlighting, hover dot transitions, fade-up reveals

v2-warm-stepwise Jun 25, 2026 Stable

Soothing step-by-step wizard with share link

A complete rethink — guided 4-step wizard (Basics → Days → Activities → Review & Share). Headspace-meets-Typeform aesthetic with warm cream tones, sage green accents, and Playfair Display headings. Generates a copyable share link on completion.

Wizard flow Share link Warm palette Playfair Display Step-by-step Guided
Update log
  • 4-step guided wizard: Trip basics → Add days → Fill activities per day → Review & share
  • Progress indicator with step dots and labels across the top
  • Vibe picker (Relaxed / Active / Foodie / Cultural / Adventure / Wellness) on step 1
  • Auto-generate days from start/end dates; manual day labels and reordering
  • Inline activity entry per day with time + name fields
  • Review page shows full itinerary grouped by day with a shareable URL mock
  • Warm cream (#faf7f2) background, sage (#7c9a7e) accents, terracotta (#c17a5e) share button
  • Pre-filled with a complete 7-day Tokyo foodie trip for instant feel
  • Fixed from initial deploy: all 7 days filled, lands on review page directly
Design stance
  • Layout: Single centered column (max 600px), no sidebar or map panel
  • Typography: Playfair Display headings + system sans-serif body
  • Color: Cream + sage + terracotta — soothing, not utilitarian
  • Interaction: Linear wizard with back/next navigation; toast feedback; copy-to-clipboard share

v1-sidebar Jun 25, 2026 Stable base

Utilitarian sidebar + map panel

Apple HIG-inspired layout with a 320px sidebar showing day-by-day timeline and a main map panel that updates with numbered pins when a day is selected. Inline activity entry with toast feedback.

Sidebar layout Apple HIG Map pins Inline entry System fonts
Update log
  • Split-pane layout: 320px sidebar (day list) + flex main panel (map area)
  • Click day to expand → map shows numbered pins for that day's activities
  • Inline add-activity form per day with name + time fields
  • Toast feedback on add, day creation, and export action
  • Export button logs full itinerary to console (mock)
  • Pre-filled with 4-day Tokyo trip: Shinjuku → Shibuya → Asakusa → Kamakura
  • System font stack, #0071e3 accent, white cards — clean Apple HIG feel
Design stance
  • Layout: Sidebar master-detail — itinerary is always visible, map is secondary
  • Typography: System fonts only (SF Pro feel)
  • Color: Apple HIG — white cards, blue accent, neutral grays
  • Interaction: All-at-once — everything visible, no wizard steps
